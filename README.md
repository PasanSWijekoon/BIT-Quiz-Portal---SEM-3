# BIT Quiz Portal - Semester 3

A web-based interactive quiz system for practicing Bachelor of Information Technology (BIT) SEM - 3 past papers from the University of Colombo School of Computing (UCSC).

## üéì Educational Purpose
This project is created strictly for **educational purposes**. It aims to help students revise and practice for their exams by providing an interactive way to attempt past paper MCQs.

The content (questions and answers) is derived from past examination papers and is used here to facilitate learning.

## üöÄ Features
- **Subject Selection:** Choose from Semester 3 subjects.
- **Year Selection:** Pick specific past paper years.
- **Interactive Quiz:**
  - Real-time feedback on answers.
  - Explanations for correct answers.
  - Score tracking.
- **Responsive Design:** Works on mobile and desktop.
